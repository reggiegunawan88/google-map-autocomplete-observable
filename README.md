# Google Map Autocomplete

Google Map with autocomplete searchbox and custom form options.

## Tech Stacks
- ReactJS v.18.2.0
- Material UI v5.11.8
- Redux v4.2.1
- Redux Observable v2.0.0
- RxJS v7.8.0

## Reference

https://github.com/mui/material-ui
